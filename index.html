O que é HTML?

Qual sua história e importância?

Para os absolutamente leigos sobre o assunto, dizer que é uma linguagem de programação, pode já dar uma ideia sobre o que seja, porém consiste de uma ideia equivocada, pois não se trata exatamente disso. É uma linguagem, mas não é de programação.

Versões

Versão 1990

O HTML foi criado em 1990 por um físico chamado Tim Berners-Lee. Na época, Berners-Lee estava trabalhando no CERN, o laboratório de física de partículas na Suíça. Ele desenvolveu o HTML como uma maneira de compartilhar documentos com seus colegas de trabalho.

Versão 1995

A primeira versão oficial do HTML foi o HTML 2.0, que foi publicado em 1995. Esta versão incluía uma série de elementos que ainda são usados hoje, como headings (títulos), links, listas e formulários.

Versão 1997

Em 1997, o HTML 3.2 foi lançado. Esta versão introduziu novos elementos, como tabelas e applets, que permitiam a incorporação de conteúdo interativo.

Versão 1999

O HTML 4.01, lançado em 1999, trouxe ainda mais melhorias e novos recursos, incluindo suporte para CSS e scripts, bem como melhor acessibilidade e internacionalização.



No entanto, a maior mudança veio em 2014 com o lançamento do HTML5. O HTML5 introduziu uma série de novos elementos e APIs que permitiram a criação de aplicações web mais ricas e interativas. Alguns dos novos elementos incluídos no HTML5 são header, footer, article, section, nav, figure, figcaption, video, audio, e muitos outros.

Além disso, o HTML5 também introduziu uma série de APIs JavaScript, como a API de Geolocalização, a API de Armazenamento Web, a API de Histórico, a API de Arrastar e Soltar, e muitas outras. Estas APIs abriram novas possibilidades para o desenvolvimento web, permitindo aos desenvolvedores criar aplicações web que poderiam rivalizar com as aplicações de desktop em termos de funcionalidade e desempenho.

É importante notar que, embora o HTML tenha evoluído ao longo dos anos, as versões anteriores do HTML não foram tornadas obsoletas pelas novas versões. Em vez disso, cada nova versão do HTML é construída sobre as versões anteriores, adicionando novos recursos e funcionalidades, mas mantendo a compatibilidade com os recursos existentes. Isso significa que o conhecimento que você adquire ao aprender HTML será relevante e útil, não importa quão avançada a tecnologia se torne.
