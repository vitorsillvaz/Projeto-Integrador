Principais Elementos do JavaScript

1. Variáveis

Usadas para armazenar dados. Existem três palavras-chave para demonstrar variáveis:

  • let: Variável que pode ser reatribuída.
  • const: Variável constante (não pode ser reatribuída).
  • var: Forma antiga de declaração de variáveis ​​(menos usadas hoje).

    • let nome = "João"; 
    const idade = 25; 
    var cidade = "São Paulo";

2. Tipos de Dados

Os principais tipos de dados em JavaScript são:

  • String: Texto ("Olá mundo!")
  • Número: Números(45, 3.14)
  • Booleano: Verdadeiro ou falso (true, false).
  • Array: Lista de valores ([1,2,3])
  • Objetivo: Coleção de pares chave-valor ({nome: "João", idade: 20})
  • Nulo: Valor nulo intencional.
  • Indefinido: Valor indefinido.

    • let texto = "Hello";
    let numero = 42; 
    let booleano = true; 
    let lista = [1, 2, 3]; 
    let objeto = { nome: "Ana", idade: 22 };

3. Operadores

  • Aritiméticos: +, -, /, %
  • Comparação: ==, ===, !=, >, <, >=, <=,
  • Lógicos: &&(e), ||(ou), !(não).

    • let soma = 5 + 3;
    let maior = 10 > 5;
    let ambosVerdadeiros = (true && false);

4. Funções

Blocos de código que podem ser reutilizados. Declaradas como functionou como arrow functions ( =>). 

    function somar(a, b) {
      return a + b;
    } 

    const subtrair = (a, b) => a - b;

5. Estruturas de Controle

  • Condicionais: ( if, else, else if): 

    • let idade = 18;
     if (idade >= 18) {
      console.log("Maior de idade"); 
     } else { 
      console.log("Menor de idade"); 
     }
  • Laços de Repetição: ( for, while, do-while):

6. Eventos

    Interação com a página HTML. Exemplo de evento click

    document.getElementById("meuBotao").addEventListener("click", function(){
      alert("Botão clicado!");
       console.log("Maior de idade");

7. Manipulação de DOM

    JavaScript pode manipular elementos HTML:

    document.getElementById("titulo").innerText = "Novo Título";
    document.querySelector(".minhaClasse").style.color = "red";

9. Promessas e Async/Await

Usado para trabalhar com operações assíncronas:

  • Promise: Resolver ou rejeitar uma operação.
  • async/await: Facilita a leitura do código assíncrono. 

    • const prom = new Promise((resolve, reject) => { 
       setTimeout(() => resolve("Sucesso!"), 1000); 
    }); 

    prom.then(resultado => console.log(resultado)); 

    async function carregarDados() { 
       const resultado = await prom; 
       console.log(resultado); 
    }

10. Módulos

    Permitem dividir o código em partes reutilizáveis. Exemplo de exportação e importação:

     //arquivo.js 
    export function saudar(nome)  
      console.log("Olá, " + nome); 
    } 

    //main.js
    import { saudar } from './arquivo.js'; 
    saudar("Pedro");

Esses são os principais elementos do JavaScript que servem como base para o desenvolvimento de qualquer aplicação web!
